# 系统维护与后台操作指南

**适用角色**：超级管理员、管理员  
**对应项目说明书**：§2 角色与权限、§3 功能模块、§4.1/§4.2 阶段

---

## 1. 登录与访问

- **后台地址**：生产环境为 `https://lexingronghe.com`（或您部署后的管理后台路径）。
- **超管默认账号**：`admin`，初始密码见部署说明或项目说明书 §6.3（**上线后务必修改**）。
- 登录后系统根据角色显示不同菜单：超管可见全部；管理员不显示「系统级维护」类功能。

---

## 2. 仪表盘

- 进入后台默认为「仪表盘」页，可查看概览信息。
- 通过左侧导航进入各功能模块。

---

## 3. 账号管理（仅超管、管理员）

- **路径**：左侧菜单 → **账号管理**。
- **功能**：
  - 查看系统内所有账号列表（教师、家长、管理员等）。
  - **新建账号**：点击「新建」或「创建账号」，填写用户名、密码、角色（教师/家长/管理员等），保存后该用户即可用该账号登录。
- **注意**：教师、家长账号创建后，需在「学生管理」或相关业务中为其关联学生或负责学生，具体以实际界面为准。

---

## 4. 首页配置（仅超管、管理员）

- **路径**：左侧菜单 → **首页配置**。
- **功能**：管理官网首页的轮播图、主视觉视频等。
  - 按页面提供的表单项填写或上传图片/视频链接。
  - 保存后官网首页将按配置展示，无需改代码。
- 配置存储在数据库 `homepage_config` 表中，与权限体系统一。

---

## 5. 文章/活动宣传页（超管、管理员、教师）

- **路径**：左侧菜单 → **文章管理**。
- **功能**：
  - **列表**：查看已发布的富文本活动宣传页。
  - **新建/编辑**：使用富文本编辑器编写图文、插入视频，支持类似公众号的排版，完成后「发布」即可。
- 发布后的文章可用于官网或指定栏目展示（以实际前端展示逻辑为准）。

---

## 6. 教案库管理（仅超管、管理员）

- **路径**：左侧菜单 → **教案库管理**（或「教学目标/四级标签」等，以实际菜单名为准）。
- **功能**：
  - 查看、编辑「领域 → 项目 → 阶段 → 长期/短期目标」的级联结构；支持按领域、项目、阶段、长期/短期、内容筛选；每页 20 条分页展示。
  - **Excel 批量导入**：按项目说明书 §9 P3 表头（领域；项目；阶段；长期编号；长期方案；短期编号；短期方案）整理 Excel，在系统中选择「Excel 导入」并上传，系统将解析并写入数据库。
  - **一键清空**：清空全部教案库目标，仅适用于尚未产生任何教案记录的初始化场景。若存在历史教案的图表数据引用，系统将拒绝清空并提示。清空前务必二次确认。
- 标签数据供教师端勾选并生成教案使用，请保证数据完整、无重复编号。

---

## 7. 教案模板管理（仅超管、管理员）

- **路径**：左侧菜单 → **模板管理**（或「教案模板」等，以实际菜单名为准）。
- **功能**：
  - 上传 Word（.docx）或 Excel（.xlsx）模板，模板中可包含占位符如 `{{studentName}}`、`{{assessmentDate}}`、`{#targets}`…`{/targets}` 等（以项目说明书 §3.2 占位符规范为准）。
  - 上传后教师端生成教案时将使用所选模板进行渲染与导出。
- 模板文件建议存放在 NAS 指定目录（如 `/SATA存储11/web/teaching/templates`），通过系统配置的 `TEMPLATE_STORAGE_PATH` 指定。

---

## 8. 超管专项：系统维护（仅超级管理员）

- **备份**：数据库定时备份脚本为 `server/scripts/backup-mysql.sh`，建议在 NAS 上配置定时任务（如每日凌晨 2 点）执行，备份目录与保留策略请与运维约定。详细说明见项目根目录 `docs/phase5-越权与部署说明.md` 第四节。
- **环境与配置**：生产环境变量通过 **`deploy/docker/.env`**（由 **`deploy/docker/.env.example`** 复制）或 docker-compose 注入，勿在代码中写死。
- **部署与编排**：使用 Docker 时，建表与索引执行顺序见 `docs/phase5-越权与部署说明.md` 第三节；首次部署需先执行建表与种子数据，再启动后端服务。

---

## 9. 常见问题

| 问题 | 建议 |
|------|------|
| 登录后看不到「账号管理」「首页配置」 | 当前账号非超管/管理员，仅教师或家长角色无法访问这些菜单。 |
| 首页配置保存后官网未更新 | 检查官网是否从接口拉取配置（如 `/api/homepage`）；清除浏览器缓存或等待前端刷新策略。 |
| 标签导入失败 | 检查 Excel 表头是否与 §9 P3 一致、编码为 UTF-8，且无合并单元格或异常字符。 |
| 模板上传后教师端看不到 | 确认模板存储路径（TEMPLATE_STORAGE_PATH）与权限正确，后端服务有读权限。 |
