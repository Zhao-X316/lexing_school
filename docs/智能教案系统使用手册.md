# 智能教案系统使用手册

**适用角色**：教师  
**对应项目说明书**：§2 教师权限、§3.2 智能教案与动态模板、§3.3 学生档案与数据可视化

---

## 1. 登录与权限说明

- 使用管理员为您创建的**教师账号**登录管理后台。
- 登录后您只能看到与本人相关的菜单：**我的学生**、**能力评估/教案生成**、**学生图表**等；不能访问「账号管理」「教案库管理」「首页配置」等管理员功能。
- 系统仅展示**您负责的学生**，无法查看或操作其他教师的学生。

---

## 2. 我的学生列表

- **路径**：左侧菜单 → **我的学生**（或「学生列表」等，以实际菜单名为准）。
- **功能**：
  - 查看分配给您的学生名单（姓名、课时等信息）。
  - 点击某名学生可进入该生的**能力评估与教案**或**数据看板**。
- 若列表为空，请联系管理员在后台为您关联学生（将学生的「负责教师」设为您的账号）。

---

## 3. 能力评估与教案生成（核心流程）

### 3.1 选择学生

- 进入**评估与教案**页面，在顶部**选择学生**下拉框中选择要生成教案的学生。
- 选择后，页面展示该生的**历史教案摘要**（教案总数、最近评估日期）及**历史教案列表**。
- 点击「**生成新教案**」按钮，进入教案生成页（需已选择学生）。

### 3.2 勾选教学目标（领域 → 项目 → 阶段 → 长期/短期目标）

- 在生成页，左侧为**级联选择**：**领域**（7 大类）→ **项目**（小类）→ **阶段**。
- 选定领域、项目、阶段后，右侧分两列展示**长期目标**与**短期目标**，带编号，可勾选。
- **7 大类（领域）**包括：认知(语言/语前)(CVP)、语言表达(EL)、小肌肉(FM)、大肌肉(GM)、模仿(视觉/动作)(VMI)、适应行为(AB)、个人自理(PSC)。
- **关联长期目标**：勾选此选项后，勾选长期目标会自动勾选其关联的短期目标（如 1-M 对应 1-a、1-b 等）；取消长期目标时，关联短期目标会同步取消。
- 可跨不同领域/项目/阶段多次选择，已选目标会累计；界面会显示已选目标数量。

### 3.3 保存、预览与下载

- **保存草稿**：将当前学生、已选目标、模板选择保存到本地，下次进入同学生时可恢复。
- **预览**：页内展示将生成的教案文本（学生名、评估日期、教师名、已选目标列表），不调用生成接口。
- **下载教案**：
  - 选择要导出的**模板**（Word 或 Excel，由管理员事先上传；可不选则使用默认 Excel 模板）。
  - 点击「**下载教案**」按钮。
  - 系统将把学生姓名、评估日期、教师姓名及您勾选的目标列表填入模板占位符，生成 `.docx` 或 `.xlsx` 文件并触发浏览器下载。
  - 下载文件名格式为：**{学生姓名}-{老师姓名}-{年月}.xlsx**（如：张三-李老师-2026年3月.xlsx）。
- **课时**：生成教案**不扣课时**。课时由上传学生打卡记录表统计上课次数进行扣除（见项目说明书 §3.4）。

### 3.4 占位符说明（供制作模板时参考）

- 基本信息：`{{studentName}}` 学生姓名；`{{assessmentDate}}` 评估日期；`{{teacherName}}` 负责教师。
- 目标列表：Word 模板中用 `{#targets}` … `{/targets}` 包裹重复区域，区域内可用 `{{number}}`（目标编号）、`{{majorCategory}}`、`{{stage}}`、`{{content}}` 等，系统按勾选数量循环输出。

## 4. 历史教案与作废

- **路径**：在**评估与教案**首页选择学生后，下方表格展示该生过往生成的教案记录。
- **功能**：
  - 查看历史评估记录、生成时间等。
  - **作废**：若某次生成的教案有误，可对该条记录执行「作废」。作废后该记录仍保留在库中（软删除），但不再参与图表统计与对外展示，便于数据追溯与审计。

---

## 5. 学生数据看板（雷达图与折线图）

- **路径**：进入某学生详情后，进入「数据看板」或「成长图表」。
- **雷达图**：展示该生**单次评估**在 7 大类上的能力剖面，便于横向对比各维度。
- **折线图**：展示该生**多次评估**随时间的成长轨迹（如 1 月、3 月、5 月等），便于纵向追踪。
- 图表仅统计**未作废**的评估（`status = active`）；作废的评估不会出现在图表中。

---

## 6. 常见问题

| 问题 | 建议 |
|------|------|
| 看不到「我的学生」或列表为空 | 联系管理员为您的账号关联学生（设置学生的负责教师为您）。 |
| 生成教案失败或下载异常 | 检查网络、确认已勾选目标；若模板报错，联系管理员检查模板占位符。 |
| 导出的 Word/Excel 排版错乱或缺少内容 | 检查是否勾选了目标、模板占位符是否与系统约定一致；若使用循环区域，确保模板中 `{#targets}`…`{/targets}` 书写正确。 |
| 无法为某学生生成教案 | 确认该学生确认为您负责（teacher_id 为您的账号）；若仍报错，请联系管理员或技术支持。 |
| 作废后能否恢复 | 当前为软删除，数据仍保留；如需恢复展示需联系管理员或通过后续版本功能支持。 |
