## 一、教案库

### 1.1 术语与列名

| 字段 | 展示名 | 说明 |
|------|--------|------|
| major_category | 领域 | 7 大类：CVP、EL、FM、GM、VMI、AB、PSC |
| sub_category | 项目 | 小类 |
| stage | 阶段 | 1–10 |
| target_type | 长期/短期 | long_term / short_term |
| target_number | 编号 | 长期编号或短期编号，可选 |
| content | 内容 | 目标内容 |

### 1.2 Excel 导入

- **表头**：领域；项目；阶段；长期编号；长期方案；短期编号；短期方案（标准格式，不可变）
- **阶段解析**：支持阿拉伯数字（1、第1阶段）、中文数字（第一阶段、个人自理第一阶段）
- **富文本**：使用 `getCellText` 避免富文本/公式单元格产生 `[object Object]`
- **去重**：同批预去重 + 导入后 SQL 去重
- **事务**：导入全程事务保护，失败回滚

### 1.3 一键清空

- **接口**：`DELETE /api/teaching-targets/clear`
- **权限**：超管、管理员
- **行为**：直接删除全部 `teaching_targets`；`assessment_details.target_id` 有 `ON DELETE SET NULL`，清空后图表仍可依赖 `major_category` 快照展示

## 二、评估与教案

### 2.1 页面结构

- **首页** `/assessments`：选学生 → 历史摘要 + 历史教案列表 → 「生成新教案」
- **生成页** `/assessments/generate`：领域 → 项目 → 阶段 级联；长期/短期双列勾选；保存草稿、预览、下载

### 2.2 编号展示

- 长期/短期目标优先使用 `target_number`，无编号时回退为序号（1. 2. 3. …）

### 2.3 课时

- 生成教案**不扣课时**；课时由打卡报表上传扣减（见项目说明书 §3.4）

## 三、模板

- **预览**：统一预览 PDF 版本
- **上传格式**：`.xlsx,.xls,.docx,.doc`
- **占位符**：`{{studentName}}`、`{{assessmentDate}}`、`{{teacherName}}`、`{{number}}`、`{{majorCategory}}`、`{{stage}}`、`{{content}}`；循环 `{#targets}`…`{/targets}`

### 3.1 IEP 教案 Excel 生成（2025-03）

- **领域归一化**：`major_category` 兼容短码（PSC）与完整标签（个人自理(PSC)）
- **IEP 固定布局**：表头第 4 行，数据从第 6 行；第 1/2/3 列分别为领域、长期目标、短期目标
- **短期目标分行**：每个短期目标占一行；领域与长期目标列纵向合并；超 7 行时自动插入行
- **样式**：以第 6 行第 1–7 列为参考，插入行应用相同格式
- **下载命名**：`{学生姓名}-{老师姓名}-{日期}.xlsx`；CORS 暴露 `Content-Disposition` 供前端读取
- **关联长期目标**：勾选/取消长期目标时，关联短期目标（按 target_number 前缀）同步勾选/取消

## 四、首页动态化

- 官网 `onMounted` 请求 `GET /api/homepage/config`，按配置渲染 hero、philosophy、facilities、why_choose_us、contact
- 接口失败时使用默认内容，不白屏
- 详见 `docs/首页动态化项目规划.md`
