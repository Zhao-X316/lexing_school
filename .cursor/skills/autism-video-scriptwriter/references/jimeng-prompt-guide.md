# 即梦提示词写法规范

## 统一风格前缀

每条图片提示词 **必须以此开头**，不可省略或改写：

```
日系动漫风格，柔和暖色调，治愈系画风，线条细腻，色彩明亮温馨，适合育儿科普的温暖氛围
```

**色彩分级与主/强调/警示色用法**见 [visual-identity.md](visual-identity.md)；提示词里写「柔和暖色背景」「关键词高亮蓝/橙」「误区用柔和灰红」等与该文件保持一致。

## 封面图提示词（抖音竖版）

封面用于信息流点击，与单帧画面提示词同样以**风格前缀开头**，并强调**竖版构图**与**标题区 + 主视觉区分层**。详细版式见 [visual-identity.md](visual-identity.md)「封面图模板」。

### 示例 A：疑问式标题 + 角色剪影

```
日系动漫风格，柔和暖色调，治愈系画风，线条细腻，色彩明亮温馨，适合育儿科普的温暖氛围，
抖音竖版9比16封面构图，上方三分之一为大字标题区留白，
标题文字「叫名字不回头，就是自闭症吗？」清晰醒目，主色调米白与浅杏色，
下方三分之二为温馨的客厅一角剪影，年轻妈妈与小男孩背影，柔和窗光，
画面简洁、留白充足，适合育儿科普账号封面
```

### 示例 B：系列感 + 信息图小标

```
日系动漫风格，柔和暖色调，治愈系画风，线条细腻，色彩明亮温馨，适合育儿科普的温暖氛围，
竖版封面，上方标题区「误区粉碎机 第3期」小字系列标 + 主标题「感统≠体育课」，
中间主视觉为可爱的简笔画感统训练小图标排列，强调色用清澈蓝点缀关键词，
底部角色不抢戏，整体清新科普风
```

图生视频若做封面动态：可写「镜头固定，标题区文字轻微浮现，主视觉区微光扫过」。

## 图片提示词结构

```
[风格前缀]，[场景/背景]，[角色动作与表情状态（若有）]，[构图与视角]，[关键元素/道具/图标]，[氛围与光线]，[画面文字（若有）]
```

各字段说明：

| 字段 | 必填 | 说明 |
|------|------|------|
| 风格前缀 | ✅ | 统一风格描述，原样复制 |
| 场景/背景 | ✅ | 具体空间：客厅、公园、医院走廊、抽象背景等 |
| 角色动作与表情 | 仅角色画面 | 只写动作和情绪状态，不描述五官。如：「微笑着蹲下」「担忧地看着孩子」 |
| 构图与视角 | ✅ | 中景/近景/全景、正面/侧面/俯视 |
| 关键元素 | ✅ | 画面中的物体、图标、道具、文字标注 |
| 氛围与光线 | 建议 | 自然光/暖光/柔和光线、安静/温馨/明亮 |
| 画面文字 | 可选 | 需要在画面中出现的文字内容 |

### 角色描述约束

用户通过即梦「角色参考」功能上传参考图保持一致性，提示词中：

- ✅ 描述：动作、表情状态、服装风格、姿势、所处位置
- ❌ 不描述：脸型、眼睛、发型、肤色等五官细节（会与参考图冲突）
- 妈妈角色统一引用为「年轻妈妈」
- 孩子角色引用为「小男孩」或「小女孩」（按本期内容选择）

## 图生视频提示词结构（短句优化）

与 [seedance-video-patterns.md](seedance-video-patterns.md) 同源：**长提示用时间轴分段，短提示用一句压缩「镜头 + 动态 + 光/氛围」**。即梦侧仍 **每条不超过 2 句**，第二句只写 **微强化** 或 **转场收束**，勿堆叠三种以上运动。

### 推荐句式（三要素压缩）

```
[镜头运动]，[画面内谁在动 / 什么在变]，[光线或氛围收束]
```

- **第一句**：必须能回答「镜头怎么动、画面里什么在动」；信息图写「谁先后亮起」，角色写「小动作 + 可选窗光/逆光」。
- **第二句（可选）**：**微强化**——含角色时优先写「面部稳定、动作缓慢连贯」中其一；纯图表可写「节奏平稳、无频闪」。

**禁止**：同一句里「推进 + 环绕 + 快速切换」等多运动叠加；与 `[视觉]` 标签冲突的复杂调度。

### 与 `[视觉]` 转场对齐（写短句时直接用）

分镜里标注了 `[视觉:…]` 时，图生视频短句优先套用下表（仍 ≤2 句）：

| `[视觉]` | 短提示词方向（选填到第一句） |
|----------|------------------------------|
| `淡入淡出` | 画面柔和淡入或前镜淡出后本镜淡入，过渡轻缓 |
| `左推右出` | 镜头缓慢横移，或画面从一侧滑入，衔接干净 |
| `缩放过渡` | 镜头缓慢推近或拉远，主体逐渐清晰或纳入全景 |
| `信息图弹入` | 镜头固定，卡片或图标依次弹入，再亮箭头或连线 |
| `模式中断` | 前零点几秒画面骤停或轻微闪切后定格，再进入稳定镜头 |

详见 [rhythm-templates.md](rhythm-templates.md)「视觉转场标签」。

### 微强化词（句末 2–6 字，择一二）

| 场景 | 可缀 |
|------|------|
| 含角色 | `面部稳定`、`动作缓慢连贯`、`柔和自然光` |
| 信息图 | `信息依次出现`、`节奏平稳` |
| 通用 | `温暖治愈`、`安静温馨` |

### 常用镜头运动

| 运动 | 适用场景 | 效果 |
|------|----------|------|
| 镜头缓慢推进 | 角色特写、情感画面 | 增强亲密感 |
| 镜头缓慢拉远 | 收尾、全景展示 | 增强叙事感 |
| 镜头固定 | 信息图、文字动画 | 突出内容 |
| 镜头轻微平移 | 场景展示、对比 | 增加流动感 |
| 镜头微微上移 | 希望感结尾 | 积极向上感 |

### 视频提示词示例（已按上式优化）

角色场景（单句三要素 + 可选第二句微强化）：
```
镜头缓慢推进，妈妈轻抚孩子头发，窗光柔和侧打在脸上。动作缓慢连贯，面部稳定。
```

信息图：
```
镜头固定，箭头与关键词依次亮起，由左到右轻微流动感。节奏平稳。
```

对比场景：
```
镜头从左侧缓慢平移到右侧，两侧画面对比逐渐展开，过渡自然。无跳切。
```

收尾温馨：
```
镜头缓慢拉远，妈妈与孩子牵手沿小路走远，阳光略变亮。温暖治愈，动作连贯。
```

步骤图：
```
镜头固定，三张步骤卡片依次从下方弹入，箭头随后亮起连接。信息依次出现。
```

`[视觉:模式中断]` 与 Hook 首镜：
```
画面轻微闪切后立刻稳定，镜头切至客厅全景固定，妈妈与孩子互动开始。面部稳定。
```

`[视觉:信息图弹入]` 强化：
```
镜头固定，圆角卡片从下方依次弹入，数据条随后亮起。节奏平稳。
```

## Seedance 等「长视频提示词」（可选）

当用户使用 **Seedance**、**可灵** 等支持**整段时间轴描述**的文生视频工具，或明确要求「按秒写满一条视频提示词」时，在保留即梦短句的同时，可按 [seedance-video-patterns.md](seedance-video-patterns.md) 撰写：

- **画幅 + 时长 + 风格** 单行声明；
- **0–X 秒 / X–Y 秒** 分段写景别、运动、光影；
- 文末 **【强化词】** 收敛画质与人脸稳定约束。

育儿科普内容请遵循该文件中的**温和与合规**说明，避免沿用参考库里的动作片/惊悚类画面。

## 即梦操作备忘

### 图片生成

1. 将图片提示词整段复制到即梦「文生图」输入框
2. 若画面含角色，同时在「角色参考」中上传对应角色的参考图
3. 抖音主发建议 **9:16 竖版**；横版备用选 16:9（与 [visual-identity.md](visual-identity.md) 一致）
4. 生成后挑选最符合画面描述的结果

### 图生视频

1. 上传选定的图片
2. 将视频提示词复制到运动描述框
3. 视频时长建议选择与该画面对应的秒数（通常 4–8 秒）
4. 生成后检查运动是否自然、角色是否变形
