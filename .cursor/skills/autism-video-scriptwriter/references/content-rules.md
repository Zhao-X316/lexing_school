# 内容改写规范

## 受众画像

- 自闭症儿童家长，部分刚确诊，情绪敏感但渴望靠谱信息
- 对自闭症有初步认知但不系统
- 期待理性、专业、有温度的解释

## 语气定位

像一位可靠的专业人士在耐心解释——理性科普为主，适度温和与共情。

**不要像**：推销员、恐吓者、鸡汤博主。

## 三大正面原则

1. **理性克制**：用事实和逻辑说话，不夸大，不过度承诺
2. **简洁清晰**：短句为主，避免长难句，重要概念用简单例子解释
3. **共情但不贩卖焦虑**：理解家长的担心，但不放大恐惧

## 禁用词与替代表达

| 禁用表述 | 推荐替代 |
|----------|----------|
| 根治 / 治愈 | 在系统干预下获得更好的发展 |
| 百分之百 / 一定 | 研究表明 / 很多案例显示 |
| 废了 / 没希望 | 目前面临一些挑战，但有改善空间 |
| 保证 / 彻底改变人生 | 有机会在早期干预下看到积极变化 |
| 不干预就完了 / 错过就晚了 | 早期干预的窗口期很重要，建议尽早开始 |
| 正常孩子 | 普通发育的孩子 / 同龄孩子 |
| 有病 / 患病 | 被诊断为 / 存在…方面的困难 |
| 治疗 (暗示可治愈) | 干预 / 训练 / 支持 |
| 必须立刻 / 马上行动 | 建议尽早咨询专业人士 |

## 积极结果的审慎表达

表达正面信息时使用相对、审慎的说法：

- 「有机会在早期干预下获得更好的发展」
- 「在配合系统训练后，很多孩子的某些方面有明显进步」
- 「研究显示，早期介入的孩子在社交和语言方面往往有更好的表现」
- 「每个孩子的进步节奏不同，但坚持科学干预通常会看到变化」

## 句式改写规则

### 缩短长句

- 原：「如果家长能够在孩子两岁之前就发现问题并且及时带孩子去医院进行评估的话」
- 改：「如果两岁前发现问题，及时评估」

### 消除绝对化

- 原：「所有自闭症孩子都会有语言问题」
- 改：「很多自闭症孩子在语言发展上面临挑战」

### 去煽动化

- 原：「你再不行动，孩子的一生就毁了！」
- 改：「早期干预很重要，建议尽早开始」

### 增加具象

- 原：「要多跟孩子互动」
- 改：「比如蹲下来和孩子平视，用简单的词配合手势来沟通」

## 平台合规红线

以下内容绝对不能出现：

- 任何医疗效果的绝对化承诺
- 暗示某种产品/课程/机构可以治愈自闭症
- 对孩子的负面标签（废了、没救了、拖累家庭等）
- 未经证实的疗法推荐（如某些民间偏方、未经循证的干预方式）
- 涉嫌歧视残障人士的表述
