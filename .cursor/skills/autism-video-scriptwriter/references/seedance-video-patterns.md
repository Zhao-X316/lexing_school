# Seedance 类视频提示词参考范式

本文件从通用 Seedance 参考提示词中提炼**可复用的结构**，供生成**整段视频提示词**或**与即梦短句并存**时使用。  
**本 Skill 面向育儿科普**：迁移时优先 **固定镜头、缓慢推进/拉远、温馨光效**；避免血腥、惊悚、过度暴力击打等示例中的影视化描写。

与 [jimeng-prompt-guide.md](jimeng-prompt-guide.md) 的关系：

- **即梦图生视频**：仍用「每条 ≤2 句」的短提示；结构与 Seedance 同源压缩为 **镜头 + 动态 + 光/氛围**，第二句仅写 **微强化** 或转场收束；并与 `[视觉]` 转场标签有对应短句表（见 jimeng「图生视频提示词结构」）。
- **Seedance 等文生视频 / 长提示**：可按下方 **时间轴 + 强化词** 写**一整条**提示词；或在输出表中增加可选列「Seedance/长视频提示词」。

---

## 核心公式

```
主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 约束
```

写作顺序习惯：**画幅与时长 → 风格一句 → 按秒分段 → 【强化词】收尾**。

---

## 规格行（开头固定信息）

| 要素 | 常见写法 |
|------|----------|
| 画幅 | `9:16竖屏`（抖音）/ `16:9横屏` |
| 时长 | 如 `15秒`，或与分镜单镜时长一致 |
| 风格 | 与账号 VIS 一致，如：日系治愈动漫、柔和暖色、电影质感（见 [visual-identity.md](visual-identity.md)） |

---

## 时间轴分段（最重要）

按 **`0–X秒：…；X–Y秒：…`** 写满全程，每段尽量包含：

| 维度 | 说明 |
|------|------|
| 景别 | 大远景 / 全景 / 中景 / 近景 / 特写；或 POV、航拍、俯拍、仰拍 |
| 主体与动作 | 谁、做什么、环境如何变 |
| 镜头运动 | 推进、拉远、固定、平移、环绕、跟拍；科普多用固定+慢推 |
| 光与环境 | 自然光、逆光、窗光、柔光；信息图可加「图标依次亮起」 |

**科普向建议**：信息图段落用「镜头固定 + 元素依次出现」；情感段落用「缓慢推进或拉远」；转场与 [rhythm-templates.md](rhythm-templates.md) 中的 `[视觉]` 标签对齐。

---

## 题材迁移（原文档七大类 → 科普用法）

| 原类型 | 可借鉴 | 科普场景 |
|--------|--------|----------|
| 唯美人像 | 光线、景别递进、逆光轮廓 | 家庭场景、家长与孩子互动 |
| 动作战斗 | 少用于本账号；仅借「慢动作、清晰节奏」 | 一般不用打斗向 |
| 电影质感 | 悬念推进、景别切换 | 误区纠正、前后对比 |
| 奇幻科幻 | 特效层次描述方式 | 信息图、比喻画面（温和奇幻） |
| 氛围风景 | 航拍、推移、地标感 | 少用；若用则作转场空镜 |
| 动物萌宠 | 可爱节奏、特写 | 可用比喻画面，注意版权与调性 |

## 【强化词】收尾（可复用类别）

集中写在文末，便于模型稳定画质与人物：

**画质**：`4K超高清`、`电影质感`、`细节丰富`

**人物稳定**（含角色镜头时）：`面部清晰稳定`、`五官一致不变形`、`人体比例自然`、`动作流畅连贯无跳帧`

**光影**：`柔和自然光`、`逆光`、`暖色调`、`电影级打光`

**氛围**：`温暖治愈`、`清新科普`、`安静温馨`

**科普专用约束**（建议常带）：`无恐怖或血腥画面`、`适合亲子观看`

## 短示例（9:16 科普向，与账号风格一致时可替换风格句）

```
9:16竖屏，8秒，日系治愈动漫风格，柔和暖色调。0-3秒：全景固定，温馨客厅，年轻妈妈与小男孩坐在地毯上玩积木，柔和窗光。3-6秒：中景，妈妈指着绘本讲解，孩子侧耳倾听，镜头缓慢推进。6-8秒：近景，孩子点头，背景适度虚化，氛围温暖。【强化词】4K超高清，面部稳定不变形，动作连贯，温暖治愈，适合育儿科普
```

## 输出时的放置方式

1. **仅即梦**：表格只填「即梦·图生视频」短句（jimeng 规范）。  
2. **用户明确要 Seedance / 可灵等长提示**：在同一画面行或附录中，追加「长视频提示词」字段，按本节 **分段 + 强化词** 写。  
3. **一镜一条**：长提示词的秒数与表格「时间段」列一致，避免总时长冲突。
